### aid mem usage
```
  aid mem usage  Monitor memory usage
            -w, --watch  Continuously monitor memory usage.
            -j, --json   Output memory usage in JSON format.
            -p, --plot   Plot total % mem usage.
-----input-----
aid mem usage -w -p
-----output-----
total: 15.4GB, used: 13.3GB (86.6%), free: 2.1GB (13.4%)
plot: used memory %
X range: [15, 44]
Y range: [86.5, 87.1]
```
