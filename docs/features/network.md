### aid network info
```
  aid network info  Show network information
            -j, --json  Output network information in JSON format.

-----input-----
aid network info -j
-----output-----
[
  {
    "name": "WiFi",
    "transmitted": 0.2,
    "received": 1.5,
    "mac": "00:00:00:00:00:00"
  }
]
```
