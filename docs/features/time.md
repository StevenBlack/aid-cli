### aid time unix
```
  aid time unix  Display unix timestamp
            -m, --milli  Output the timestamp as unix milliseconds.

-----input-----
aid time unix 
-----output-----
1729546694
```          

### aid time dt
```
  aid time dt  Display the datetime
            -l, --local  Use the local datetime.
            -r, --rfc   Output the datetime in Rfc3339 format.

-----input-----
aid time dt
-----output-----
2024-10-21 21:38:34
```
